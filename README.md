# ServiceNow-Automation-Project-for-Network-Requests

##  Project Overview
This project focuses on automating network-related service requests using **ServiceNow Flow Designer**.  
It reduces manual intervention by handling approvals, assignments, and status updates automatically, improving overall efficiency and response time.

---

##  Key Features
-  Automated request creation through Service Catalog  
-  Approval workflow for VPN requests  
-  Auto assignment to Network Team / IT Support  
-  Real-time status tracking (Requested → Approved → Completed)  
-  Email notifications at each stage  

---

##  Technologies Used
- ServiceNow  
- Flow Designer  
- Service Catalog  
- Workflow Automation  
- JavaScript (Server-side scripting)
